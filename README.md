# ModioZodio MoonLoader Pack

Набор MoonLoader-скриптов для Arizona RP с игровым менеджером установки, обновлений и удаления.

Главная идея простая: игрок открывает менеджер в игре, видит список скриптов, читает понятное описание, ставит нужное одной кнопкой и сразу понимает, какая версия установлена.

## Менеджер

Файл менеджера: `modio_manager.lua`

Команды:

```text
/modio
/mscripts
```

В менеджере видно:

- установлен скрипт или нет;
- локальная версия;
- актуальная версия на GitHub;
- дата последнего обновления;
- версия самого менеджера и доступность его обновления;
- описание, команды, принцип работы и особенности каждого скрипта;
- кнопки установки, обновления и удаления;
- кнопка обновления самого менеджера.

## Скрипты

### Lavaka

Помощник установки лавки через CEF-цепочку Arizona RP.

Команды:

```text
/lavaka
/lavakadebug
```

Скрипт открывает интерактивное меню, ждет ответ игры, отправляет действие установки лавки и выключается после результата. Фоновый CEF-перехват убран, чтобы не ломать другие игровые CEF-окна.

### CtrlLKM

Helper для автоматического Ctrl + ЛКМ.

Команда:

```text
/ctrllkm
```

Удерживает Ctrl, циклически нажимает ЛКМ и выключается, если игрок сам нажал Ctrl или в чате появилось сообщение про легендарный приз.

### FPSFix

FPSFix с дополнительными инструментами и встроенным разделом Lavaka.

Команда:

```text
/fps
```

Подходит тем, кто хочет запускать помощник лавки из общего окна FPSFix.

### InfoZZ

Внутриигровой TXT-справочник.

Команда:

```text
/infozz
```

Позволяет читать TXT-файлы, искать по ним и использовать локальный AI-поиск по базе.

## Как работает обновление

Менеджер читает актуальный список скриптов из:

```text
https://raw.githubusercontent.com/ilhamVode/moonloader-pack/main/manifest.json
```

Если версия на GitHub отличается от локальной версии в файле скрипта, менеджер покажет, что доступно обновление.

## Самообновление менеджера

Менеджер также проверяет собственную версию через блок `manager` в `manifest.json`.

Если на GitHub опубликована новая версия `modio_manager.lua`, в окне можно нажать `Обновить менеджер`. Файл скачивается во временную папку, текущая версия временно переносится в `.bak`, после чего новая версия ставится на место.

После успешного обновления нужно нажать `Перезагрузить Lua`, чтобы MoonLoader запустил уже новую версию менеджера.

## Установка менеджера

1. Скачайте `modio_manager.lua`.
2. Поместите файл в папку `moonloader`.
3. Запустите игру или перезагрузите MoonLoader-скрипты.
4. В игре откройте `/modio`.

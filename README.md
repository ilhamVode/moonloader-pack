## 🚀 Быстрый старт

1. **Скачай** [`modio_manager.lua`](https://github.com/ilhamVode/moonloader-pack/releases/tag/v1.8.3.4) — на странице релиза нажми на файл `modio_manager.lua` в списке Assets
2. **Перемести** файл в папку `GTA San Andreas\moonloader\`
3. **Запусти игру** (или перезагрузи MoonLoader: `Ctrl+R`)
4. **Открой менеджер** командой в чате:

```
/modio   (или /mscripts)
```

Менеджер сам загрузит каталог скриптов с GitHub — выбирай, устанавливай, обновляй.

### 🔹 Lavaka

<!-- СКРИНШОТ LAVAKA -->
<p align="center">
  <i>(скриншот работы Lavaka)</i>
</p>

Помощник установки лавки через интерактивное CEF-меню Arizona RP. Скрипт не кликает вслепую: он отправляет запрос, ждёт ответ игры, выполняет действие установки и выключается после результата.

**Команды:**
```
/lavaka       — включить или выключить помощник установки лавки
/lavakadebug  — включить или выключить диагностические сообщения
```

**Как пользоваться:** Встань на место установки лавки и включи `/lavaka`. Скрипт будет сам повторять попытки с учётом игровых задержек. После успешной установки или сообщения, что лавка уже установлена, помощник остановится.

**Особенности:**
- Строгая CEF-цепочка: запрос меню → ожидание ответа → действие установки → ожидание закрытия окна
- Адаптивное ожидание под игровые ограничения
- Сообщение о примерном времени установки
- Защита от ложных сообщений игроков: реакция только на системные подсказки и ошибки

> Фоновый CEF-перехват убран специально: так скрипт не вмешивается в другие CEF-окна игры, например `/time`.

### 🔹 CtrlLKM

<!-- СКРИНШОТ CTRLLKM -->
<p align="center">
  <i>(скриншот работы CtrlLKM)</i>
</p>

Helper для автоматического Ctrl + ЛКМ. Подходит для действий, где нужно удерживать Ctrl и часто нажимать левую кнопку мыши.

**Команды:**
```
/ctrllkm  — включить или выключить Ctrl + ЛКМ helper
```

**Как пользоваться:** Запусти `/ctrllkm`. Скрипт удерживает Ctrl и нажимает ЛКМ по циклу. Если ты сам нажмёшь Ctrl или нажмёшь ESC, скрипт сразу выключится. Также он отключается, если игра сворачивается или становится неактивной.

**Особенности:**
- Автоматическое удержание Ctrl
- Циклическое нажатие ЛКМ
- Отключение при ручном нажатии Ctrl
- Отключение по сообщению в чате с текстом "и выиграл легендарный приз"
- Без повторяющихся подсказок в чат во время работы

### 🔹 InfoZZ

<!-- СКРИНШОТ INFOZZ -->
<p align="center">
  <i>(скриншот работы InfoZZ)</i>
</p>

Внутриигровой TXT-справочник для SA:MP. Читает локальные материалы, умеет искать по ним и скачивать готовую TXT-базу с GitHub.

**Команды:**
```
/infozz  — открыть справочник
```

**Как пользоваться:** Открой `/infozz`, выбери TXT-файл слева и используй поиск. Кнопка "Скачать с GitHub" подтягивает опубликованную базу из Modio Pack в `moonloader\config\infozz`.

**Особенности:**
- Просмотр TXT-файлов в игровом окне
- Поиск по загруженным материалам
- Локальный AI-поиск по базе
- Скачивание готовых TXT-файлов с GitHub без Python
- Обновление списка файлов без перезапуска игры

> Python-парсер используется только разработчиком для подготовки TXT перед публикацией. Обычному игроку нужен только InfoZZ.

### 🔹 AutoOpenRoulettes

<!-- СКРИНШОТ AUTOOPENROULETTES -->
<p align="center">
  <i>(скриншот работы AutoOpenRoulettes)</i>
</p>

Автоматическое открытие поддерживаемых рулеток Arizona RP через CEF-окно CrateRoulette. Скрипт запускает открытие, забирает приз и выходит из окна рулетки.

**Команды:**
```
/autoroulette  — включить или остановить автооткрытие рулеток
```

**Как пользоваться:** Открой окно рулетки в игре. Когда скрипт определит поддерживаемую рулетку, используй `/autoroulette` для запуска. Призы записываются в папку `moonloader\Roulette` по датам.

**Особенности:**
- Поддерживаемые id рулеток: 555, 556, 557, 1425
- Автоматически отправляет `crate.roulette.open`
- После открытия забирает приз через `crate.roulette.takePrize`
- Закрывает окно рулетки через `crate.roulette.exit`
- Логирует полученные призы в текстовый файл

_Автор: CaJlaT_

### 🔹 A-Doklad

<!-- СКРИНШОТ A-DOKLAD -->
<p align="center">
  <i>(скриншот работы A-Doklad)</i>
</p>

Скрипт для автоматических докладов в рацию на посту для армий Arizona RP. Отправляет заданный доклад через выбранный интервал и помогает не забывать регулярные сообщения на посту.

**Команды:**
```
/dmenu      — открыть меню настройки доклада
/doklad     — включить автоматические доклады или показать время до следующего доклада
/dokladoff  — выключить автоматические доклады
```

**Как пользоваться:** Открой `/dmenu`, укажи имя, пост, состояние и интервал. После `/doklad` скрипт отправляет доклад в рацию и повторяет его через заданное время. При включённых настройках может дополнительно вызвать `/time` и сделать F8.

**Особенности:**
- Настройка имени, поста и состояния
- Отправка доклада в `/r` по заданному интервалу
- Команда для проверки времени до следующего доклада
- Опциональный вызов `/time`
- Опциональное нажатие F8 после доклада
- Сохранение настроек в `doklad_settings.ini`

_Автор: Caps_

### 🔹 Arenda Helper

<!-- СКРИНШОТ ARENDA HELPER -->
<p align="center">
  <i>(скриншот работы Arenda Helper)</i>
</p>

Помощник сдачи аксессуаров и предметов в аренду на серверах Arizona RP. Позволяет быстро настроить ID предмета, цену, срок аренды и запускать сдачу через удобное mimgui-окно.

**Команды:**
```
/arenda  — открыть главное меню управления
```

**Как пользоваться:** Открой `/arenda`, выбери режим аренды, укажи ID предмета, цену и срок. Для массовой сдачи можно заранее собрать сет аксессуаров и запускать его в пару кликов.

**Особенности:**
- Режимы аренды: по дням, по часам, до ближайшего рестарта, на максимальный срок
- Система сетов для групп аксессуаров
- Автоматизация диалогов инвентаря
- Показ ID предметов прямо в инвентаре
- Тихий режим для уменьшения сообщений в чате
- Сохранение настроек в `config/arenda.json`

_Автор: Baretti_

### 🔹 News Helper

<!-- СКРИНШОТ NEWS HELPER -->
<p align="center">
  <i>(скриншот работы News Helper)</i>
</p>

Помощник для сотрудников СМИ на Arizona RP. Упрощает редакцию объявлений, эфиры, собеседования, экзамены и ведёт статистику работы.

**Команды:**
```
/rnh  — открыть меню News Helper
```

**Как пользоваться:** Открой `/rnh` и выбери нужный раздел: редакция объявлений, статистика, эфиры, собеседование, экзамены или настройки. Скрипт также умеет искать похожие объявления среди уже отредактированных.

**Особенности:**
- Помощь в редакции объявлений: автозамены, окно помощи и горячие клавиши
- Статистика заработка и количества отредактированных объявлений
- Разделы для эфиров, собеседований и экзаменов
- Поиск похожих объявлений по истории редакции
- Автоматическое скачивание нужного шрифта
- Настраиваемая тема интерфейса

_Автор: fa1ser_

### ⚠️ Запрещённые скрипты

> **Внимание:** Эти скрипты могут привести к бану. Используй только если понимаешь риск.

#### 🔴 FPSFix

FPSFix с дополнительными инструментами, встроенным разделом Lavaka и автоедой по CEF-событию сытости Arizona HUD.

**Команды:**
```
/fps  — открыть окно FPSFix
```

**Особенности:**
- Основные функции FPSFix
- Встроенный помощник установки лавки с логикой отдельного `lavaka.lua`
- Окно закрывается при запуске установки лавки, чтобы не мешать курсору
- Отображение последнего результата установки
- Автоеда срабатывает по `event.arizonahud.playerSatiety` при сытости 20 или ниже

> Авторская основа FPSFix сохранена. Автоеда читает только точечное CEF-событие `playerSatiety` и не блокирует другие CEF-пакеты.

_Автор: JustFedot / ModioZodio_

---

#### 🔴 Gribi

Рендер-помощник для поиска грибов. Скрипт ищет 3D-текст "Срезать гриб" и рисует на экране метку с дистанцией до гриба.

**Команды:**
```
/gribs  — включить или выключить рендер грибов
```

**Особенности:**
- Поиск 3D-текста "Срезать гриб"
- Рендер подписи на экране поверх позиции гриба
- Показ дистанции до гриба в метрах
- Переключение одной командой без дополнительного окна

_Автор: vlaDICK2288_

---

#### 🔴 bFishing

Помощник для рыбалки на Arizona RP. Автоматизирует прохождение новой CEF-мини-игры, умеет заново забрасывать удочку после улова и позволяет настраивать задержки действий.

**Команды:**
```
/bfish  — открыть или закрыть меню настроек
```

**Особенности:**
- Автоматическое прохождение CEF-мини-игры рыбалки
- Автозаброс через `/fishrod` после сообщения об улове
- Настраиваемые фиксированные или случайные задержки
- Выбор слота удочки в диалоге
- Сохранение настроек в `moonloader/config/bFishing.json`

> Требования: MoonLoader 0.26+, mimgui, inicfg, arizona-events, samp.events, encoding.

_Автор: bakhusse_

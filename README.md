# ModioZodio MoonLoader Pack

Набор MoonLoader-скриптов для Arizona RP с игровым менеджером установки и обновлений.

## Что внутри

- `modio_manager.lua` - менеджер скриптов с окном в игре.
- `lavaka.lua` - помощник установки лавки.
- `ctrllkm.lua` - Ctrl + LKM helper.
- `FPSFix.lua` - FPSFix с дополнительными инструментами.

## Возможности менеджера

- показывает локальную версию скрипта;
- показывает актуальную версию из `manifest.json`;
- показывает дату последнего обновления на GitHub;
- умеет устанавливать, обновлять и удалять скрипты прямо из игры;
- сообщает, установлена ли последняя версия;
- использует raw-файлы GitHub без стороннего лаунчера.

## Установка

1. Скачайте `modio_manager.lua`.
2. Переместите файл в папку:

```text
moonloader
```

3. Запустите игру или перезагрузите MoonLoader-скрипты.
4. В игре откройте менеджер командой:

```text
/modio
```

Также доступна команда:

```text
/mscripts
```

## Обновления

Менеджер читает актуальный список скриптов из:

```text
https://raw.githubusercontent.com/ilhamVode/moonloader-pack/main/manifest.json
```

Если версия на GitHub отличается от локальной версии, в окне появится статус обновления.

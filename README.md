## 🚀 Быстрый старт

1. **Скачай** [`modio_manager.lua`](https://github.com/ilhamVode/moonloader-pack/blob/main/modio_manager.lua) — открой страницу и нажми **Download raw file** (или правой кнопкой по ссылке → "Сохранить как...")
2. **Перемести** файл в папку `GTA San Andreas\moonloader\`
3. **Запусти игру** (или перезагрузи MoonLoader: `Ctrl+R` в чате)
4. **Открой менеджер** командой в чате:

```
/modio   (или /mscripts)
```

Готово. Менеджер сам загрузит каталог скриптов с GitHub — выбирай, устанавливай, обновляй.

| | |
|---|---|

## 📜 Доступные скрипты

### Lavaka
Помощник установки лавки через CEF-меню. `/lavaka`

### CtrlLKM
Автоматический Ctrl + ЛКМ. `/ctrllkm`

### InfoZZ
Внутриигровой TXT-справочник с поиском. `/infozz`

### AutoOpenRoulettes
Автооткрытие рулеток Arizona RP. `/autoroulette`
<br>_Автор: CaJlaT_

### A-Doklad
Автодоклады в рацию для армий. `/dmenu`

### Arenda Helper
Сдача предметов в аренду. `/arenda`
<br>_Автор: Baretti_

### News Helper
Помощник для сотрудников СМИ. `/rnh`
<br>_Автор: fa1ser_

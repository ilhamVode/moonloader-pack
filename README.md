# ModioZodio MoonLoader Pack

Набор MoonLoader-скриптов для Arizona RP с игровым менеджером установки, обновления и удаления.

## Modio Manager

Файл менеджера: `modio_manager.lua`

Команды:

```text
/modio
/mscripts
```

Менеджер загружает каталог из:

```text
https://github.com/ilhamVode/moonloader-pack/raw/HEAD/manifest.json
```

В менеджере есть список скриптов, версии, описания, команды, история обновлений, установка, обновление, удаление и самообновление менеджера.

## Скрипты

### Lavaka

Помощник установки лавки через интерактивное CEF-меню Arizona RP.

Команды:

```text
/lavaka
/lavakadebug
```

Скрипт отправляет запрос на открытие интерактивного меню, ждет ответ игры, выполняет действие установки лавки и выключается после результата. Фоновый CEF-перехват убран, чтобы не ломать другие игровые CEF-окна.

### CtrlLKM

Helper для автоматического Ctrl + ЛКМ.

Команда:

```text
/ctrllkm
```

Удерживает Ctrl, циклически нажимает ЛКМ и выключается, если игрок сам нажал Ctrl или в чате появилось сообщение про легендарный приз.

### FPSFix

FPSFix с дополнительными инструментами, встроенным разделом Lavaka и автоедой по CEF-событию сытости Arizona HUD.

Статус: запрещенный. Есть возможность получить бан.

Команда:

```text
/fps
```

Автоеда читает `event.arizonahud.playerSatiety` и выполняет команду еды, когда сытость становится `20` или ниже.

### InfoZZ

Внутриигровой TXT-справочник.

Команда:

```text
/infozz
```

Позволяет читать TXT-файлы, искать по ним и использовать локальный AI-поиск по базе.

Кнопка `Скачать с GitHub` загружает опубликованную базу TXT из pack в `moonloader\config\infozz`. Python-парсер нужен только для подготовки этих файлов перед публикацией, обычному игроку он не требуется.

### AutoOpenRoulettes

Автоматическое открытие поддерживаемых рулеток Arizona RP.

Автор: `CaJlaT`

Команда:

```text
/autoroulette
```

Работает через CEF-окно `CrateRoulette`: запускает открытие, забирает приз, закрывает окно и записывает полученные призы в `moonloader\Roulette` по датам.

### Gribi

Рендер-помощник для грибов.

Автор: `vlaDICK2288`

Статус: запрещенный. Есть возможность получить бан.

Команда:

```text
/gribs
```

Ищет 3D-текст `Срезать гриб`, рисует метку `Грибок` и показывает дистанцию до найденного гриба.

### bFishing

Помощник для рыбалки на Arizona RP.

Автор: `bakhusse`

Статус: запрещенный. Есть возможность получить бан.

Команда:

```text
/bfish
```

Автоматизирует новую CEF-мини-игру рыбалки, умеет заново забрасывать удочку после улова и сохраняет настройки в `moonloader/config/bFishing.json`.

### A-Doklad

Скрипт для автоматических докладов в рацию на посту для армий Arizona RP.

Автор в pack: `Caps`

Команды:

```text
/dmenu
/doklad
/dokladoff
```

Отправляет заданный доклад через выбранный интервал и хранит настройки в `doklad_settings.ini`.

### Arenda Helper

Помощник сдачи аксессуаров и предметов в аренду.

Автор: `Baretti`

Команда:

```text
/arenda
```

Поддерживает аренду по дням, по часам, до рестарта и на максимальный срок. Есть система сетов, показ ID предметов в инвентаре, тихий режим и сохранение настроек в `config/arenda.json`.

### News Helper

Помощник для сотрудников СМИ на Arizona RP.

Автор: `fa1ser`

Команда:

```text
/rnh
```

Помогает редактировать объявления, вести статистику, проводить эфиры, собеседования и экзамены. Также умеет искать похожие объявления по истории редакции.

## Установка

1. Скачайте `modio_manager.lua`.
2. Поместите файл в папку `moonloader`.
3. Запустите игру или перезагрузите MoonLoader-скрипты.
4. В игре откройте `/modio`.
